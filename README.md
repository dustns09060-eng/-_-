## 포함 기능
- 링크 접속 시 접속 비밀번호 확인
- 앱잠금 TRUE일 때 앱잠금 비밀번호 확인
- 맞팔잠금 TRUE일 때 맞팔확인 비밀번호 확인
- 운영진 비밀번호를 Apps Script에서 실시간 확인
- 운영진에서 앱잠금 ON/OFF
- 운영진에서 맞팔잠금 ON/OFF
- 접속·앱잠금·맞팔 비밀번호 변경
- 공지 작성·삭제
- Sheet1 명단 실시간 불러오기
- 500명씩 보기, 검색, 인스타 바로가기
- 단톡방 기준 맞팔 분석
- 결과 복사 및 CSV 다운로드
- 카카오톡 최소 미리보기 유지
