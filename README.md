# 여우방 팔로우리스트+맞팔확인 V32 POLISHED UI

## 변경 사항
- 상단 헤더를 `여우방`만 보이도록 단순화
- 팔로우리스트 검색창과 조 선택 영역을 스크롤 시 상단 고정
- 닉네임과 아이디는 한 줄 표시 유지
- `열기 →`를 `↗ 열기`로 변경
- 리스트 행 디자인과 간격 정리
- 운영진 설정 항목을 카드형으로 정돈
- 하단 메뉴 터치 효과와 그림자 개선
- 기존 접속 분리, 앱잠금, 맞팔잠금, 공지, 실시간 연동 기능 유지
