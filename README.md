# 여우방 팔로우리스트+맞팔확인 V28 MINIMAL ACCESS

## 변경 사항
- 첫 화면을 `일반 접속`과 `운영진 접속`으로 분리
- 앱잠금 TRUE일 때 일반 접속은 차단
- 운영진은 앱잠금 상태에서도 운영진 비밀번호로 접속 가능
- 일반 접속자는 운영진 하단 메뉴가 보이지 않음
- 운영진 접속자는 운영진 메뉴가 표시되고 바로 운영진 화면으로 이동
- 팔로우리스트와 운영진의 불필요한 설명 문구 삭제
- 통계 카드 최소화
- 새로고침 버튼을 아이콘 형태로 단순화
- 앱잠금 비밀번호 변경 UI 제거
- 기존 실시간 동기화, 공지, 맞팔확인, 로그 기능 유지
