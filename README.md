# 여우방 맞팔확인 V16 SIMPLE

GitHub Pages에 바로 배포할 수 있는 완성본입니다.

## 주요 기능
- Google Sheets 단톡방 명단 자동 연동
- 인스타그램 ZIP 업로드 및 자동 분석
- 단톡방 사람만 비교
- 맞팔 완료 / 나만 팔로우 함 / 상대가 팔로우만 함 / 서로 팔로우 안 함
- 아이디·닉네임 검색
- 인스타그램 바로가기
- 미맞팔 명단 복사
- CSV 다운로드
- PWA 홈 화면 설치

## GitHub 업로드
ZIP을 푼 뒤 안의 파일 전체를 GitHub 저장소 루트에 업로드하세요.

## 시트 변경
`config.json`의 `sheetId`와 `sheetName`을 수정하면 됩니다.
