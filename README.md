# 여우방 맞팔확인 V15 SIMPLE

단톡방 명단에 있는 사람만 인스타그램 맞팔 여부를 확인하는 심플 버전입니다.

## 기능
- 인스타그램 ZIP 업로드
- Google Sheets 단톡방 명단 자동 연동
- 맞팔 완료 / 내가 팔로우만 함 / 상대가 팔로우만 함 / 서로 팔로우 안 함
- 아이디·닉네임 검색
- 인스타 바로가기
- 현재 명단 복사
- CSV 다운로드

## 업로드
ZIP 안의 파일 전체를 GitHub 저장소 루트에 덮어쓰기 업로드하세요.

## 명단 설정
`config.json`에서 `sheetId`, `sheetName`을 수정할 수 있습니다.
